# K个一组翻转链表

[题目请看这里](https://leetcode-cn.com/problems/reverse-nodes-in-k-group/)

## 描述

给定K值，然后依次对链表的k个元素进行翻转

如果剩余不满K个，则不进行翻转

## 思路

根据翻转链表的思路，再进行计数，每遇到k个元素，就翻转一次

思路并不算复杂，主要是编码较困难

## 代码

封装翻转链表的方法

```Java
class Solution {
    public ListNode reverseKGroup(ListNode head, int k) {
        if (k == 1) return head;
        // 初始化
        ListNode dummyHead = new ListNode();
        dummyHead.next = head;
        ListNode cur = head;
        ListNode tail = dummyHead;
        int count = 1;

        while (cur != null) {
            if (count == k) {
                ListNode next = cur.next;
                cur.next = null;
                tail = reverseList(tail.next, tail, next);
                count = 1;
                cur = next;           
            } else {
                count ++;
                cur = cur.next;
            }
        }
        return dummyHead.next;
    }

    /*
    * 举例：如果链表为 1-> 2->3 ->4 ，要翻转 2->3
    * @description    返回翻转后的尾部，或者说翻转前的头部	2    
    * @param head     要翻转部分的头节点	2
    * @param preHead  翻转链表外的头部	1
    * @param preTail  翻转链表外的尾部	4
    */
    public ListNode reverseList(ListNode head, ListNode preHead, ListNode preTail) {
        ListNode cur = head;
        ListNode tmp = cur.next;
        while (true) {
            tmp = cur.next;
            if (tmp == null) break;
            cur.next = preTail;
            preTail = cur;
            cur = tmp;    
        }
        cur.next = preTail;
        preHead.next = cur;
        return head;
    }
}
```
