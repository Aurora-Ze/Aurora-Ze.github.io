# 排序数组

[题目请看这里](https://leetcode-cn.com/problems/sort-an-array/)

## 描述

对给定数组进行排序

## 思路

使用排序算法

如**归并排序、快速排序、堆排序**等

## 代码1：归并排序

主要过程

1. 对数组进行划分
2. 划分到长度小于等于7时，用插入排序
3. 然后对两个排序好的数组，进行合并

```Java
class Solution {
    public int[] sortArray(int[] nums) {
        int[] temp = new int[nums.length];
        mergeSort(nums, 0, nums.length - 1, temp);
        return nums;
    }

    public void mergeSort(int[] nums, int left, int right, int[] temp) {
        if (right - left < 7) {
            insertSort(nums, left, right);
            return;
        }
        int mid = (right + left) >>> 1;
        mergeSort(nums, left, mid, temp);
        mergeSort(nums, mid + 1, right, temp);
        // 判断划分完毕的数组是否已经有序
        if (nums[mid] <= nums[mid+1]) {
            return;
        }

        merge(nums, left, right, mid, temp);
    }

    public void insertSort(int[] nums, int left, int right) {
        for (int i = left + 1; i <= right; ++ i) {
            int tmp = nums[i];
            int j = i;
            while (j > left && tmp < nums[j-1]) {
                nums[j] = nums[j-1];
                j--;
            }
            nums[j] = tmp;
        }
    }
    public void merge(int[] nums, int left, int right, int mid, int[] temp) {
        System.arraycopy(nums, left, temp, left, right - left + 1);
        int i = left;
        int j = mid + 1;
        for (int k = left; k <= right; ++ k) {
            if (i == mid + 1) {
                nums[k] = temp[j];
                j ++;
            } else if (j == right + 1 || temp[i] <= temp[j]) {
                nums[k] = temp[i];
                i ++;
            } else {
                nums[k] = temp[j];
                j ++;
            }
        }
    }
}
```
