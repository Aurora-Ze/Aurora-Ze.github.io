## 状态模式

#### 描述

允许对象在内部状态改变时改变它的行为，对象看起来好像修改了它所属的类。

**主要元素**

- Context

  拥有多个具体状态对象和一个当前状态对象，在调用状态对象的方法时，如果状态发生改变，则修改当前状态对象。

- State

  状态接口

- ConcreteState

  具体的状态对象
