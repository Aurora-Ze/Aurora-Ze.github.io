# 合并两个有序数组

[88. 合并两个有序数组 - 力扣（LeetCode） (leetcode-cn.com)](https://leetcode-cn.com/problems/merge-sorted-array/)

### 描述

给定两个`有序数组`nums1和nums2，把两个数组合并成nums1数组。

假定nums1数组容量够大

### 思路

从nums1数组的最后开始填充，并用双指针来标记取元素的位置。

### 代码

```Java
public void merge(int[] nums1, int m, int[] nums2, int n) {
        // 从有序数组末尾开始向前遍历
        int p = m - 1;
        int q = n - 1;

        int end = m + n - 1;
        while (q != -1  && p != -1) {
            if (nums1[p] < nums2[q]) {
                nums1[end] = nums2[q];
                q --;
            } else {
                nums1[end] = nums1[p];
                p --;
            }
            end --;
        }
        while (p != -1) {
            nums1[end] = nums1[p];
            end --;
            p --;
        }
        while (q != -1) {
            nums1[end] = nums2[q];
            end --;
            q --;
        }
    }
```
